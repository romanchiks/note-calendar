# Запуск проекта

## Установка

Скачать проект и установить зависимости:

```bash
git clone https://github.com/romanchiks/note-calendar

# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Запуск тестового сервера

В результате страница будет доступна по адресу `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
